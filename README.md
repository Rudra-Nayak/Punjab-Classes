# EliteJEE Classes Landing Page

A premium, highly interactive, and SEO-optimized responsive landing page for **EliteJEE Classes** — a premier coaching institute for JEE Mains & Advanced preparation.

## Features

- **Cinematic Hero Section**: Engaging design, custom trust counters, and dynamic shimmering Call-to-Action buttons.
- **Detailed Instructor Bio**: Focus on Gurwinder Singh Sir's 21+ years of senior faculty experience at ALLEN, AAKASH, and BANSAL, showcasing credentials and student stats.
- **Why Choose Us Section**: Elegant feature cards displaying key advantages (small batch size, expert faculty, live classes, doubt clearing, etc.) with custom hover micro-animations.
- **Interactive Batch Schedule**: Toggle between Class 11 and Class 12 batches with real-time seat availability indicators.
- **Student Success & Testimonials**: Carousel displaying high-scoring student testimonials and reviews.
- **Premium Enrollment Form**: Custom input fields, validation, and alternate quick contact via WhatsApp.
- **Fully Responsive & Accessible**: Built with mobile-first semantic HTML, keyboard-navigable components, and glassmorphic designs.
- **Advanced SEO & Schema**: Implemented JSON-LD EducationalOrganization structured data, meta descriptions, and open graph cards for search optimization.

## Tech Stack

- **Core**: Next.js 16 (App Router), React 19, TypeScript
- **Styling**: Tailwind CSS v4, PostCSS, Custom Vanilla CSS Animations
- **Icons**: Lucide React
- **Dev Tools**: ESLint

## Getting Started

First, install dependencies:

```bash
npm install
```

Second, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## Building for Production

To create an optimized production build:

```bash
npm run build
```

To run the production build:

```bash
npm run start
```
